# YouTube Shorts Hide Script

A browser bookmarklet that hides YouTube Shorts from your timeline across all of your devices—not just in your current browser.

## Installation

### 1. Download the Bookmarklet

Go to the [latest release](../../releases/latest) and download the `inline.js` file.

### 2. Add to Your Browser

#### Chrome / Edge / Brave
1. Open the `inline.js` file in a text editor
2. Copy the entire contents
3. Right-click your bookmarks bar and select **Add page** (or **Add bookmark**)
4. Name it "Hide YouTube Shorts"
5. Paste the copied code into the **URL** field
6. Click **Save**

#### Firefox
1. Open the `inline.js` file in a text editor
2. Copy the entire contents
3. Right-click your bookmarks bar and select **New Bookmark**
4. Name it "Hide YouTube Shorts"
5. Paste the copied code into the **Location** field
6. Click **Add**

#### Safari
1. Open the `inline.js` file in a text editor
2. Copy the entire contents
3. Show your bookmarks bar (View → Show Favorites Bar)
4. Right-click the bookmarks bar and select **Add Bookmark**
5. Name it "Hide YouTube Shorts"
6. Paste the copied code into the **Address** field
7. Click **Add**

## Usage

1. Go to YouTube subscriptions page
2. Click the "Hide YouTube Shorts" bookmark in your bookmarks bar
3. Shorts will be hidden from your timeline across all devices

## How It Works

This bookmarklet doesn't just hide Shorts visually in your browser—it actually interacts with YouTube to hide them from your account, so they stay hidden across all your devices.
